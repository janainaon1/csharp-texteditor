## Running the Samples From the Command Line

1. Clone this repository:
```bash 
git clone https://github.com/janainaon1/csharp-texteditor.git
```

2. Entry the folder:
```bash 
cd csharp-texteditor
```

3. Run the application:
```bash 
dotnet run 
```
